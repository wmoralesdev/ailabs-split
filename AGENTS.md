# AGENTS.md

## Cursor Cloud specific instructions

This is a single-package **TanStack Start** app (React 19 + TypeScript + Vite 8 + Tailwind v4 + shadcn/ui), managed with **pnpm**. There is no backend, database, or external service — the only runtime service is the Vite dev server. No environment variables or secrets are required.

Standard commands live in `package.json` (`scripts`):

- Dev server: `pnpm dev` → serves on port **3000** (`vite dev --port 3000`). This is the only service to run.
- Build: `pnpm build`; preview built output: `pnpm preview`.
- Typecheck: `pnpm typecheck` (passes clean).
- Tests: `pnpm test` (`vitest run`). There are currently no test files, so it exits non-zero with "No test files found" — this is expected, not a failure.
- Lint: `pnpm lint`. The scaffolded template ships with 2 pre-existing `import/consistent-type-specifier-style` errors in `src/components/ui/button.tsx` and `src/lib/utils.ts`; these predate any changes. `pnpm lint --fix` resolves them.

Non-obvious gotchas:

- `src/routeTree.gen.ts` is auto-generated by the TanStack Router plugin and is regenerated (with single-quote style that differs from the committed double-quote version) whenever the dev server or build runs. Do not commit these cosmetic regenerations; run `git checkout src/routeTree.gen.ts` if it shows up dirty.
- HMR works out of the box: editing files under `src/` live-updates the browser without a manual refresh.
